# Company AI Chat - User Guide

## Introduction

Welcome to the Company AI Chat! This tool provides a secure, private, and powerful AI assistant experience powered by Azure OpenAI.

## Accessing the Application

- **URL**: [https://vesper-chat.eastus.azurecontainerapps.io](https://vesper-chat.eastus.azurecontainerapps.io) _(Note: This link will be active once deployed)_
- **Local URL**: [http://localhost:3000](http://localhost:3000) _(For testing)_

## Logging In

1.  Navigate to the URL.
2.  Click the **"Sign in with Microsoft"** button.
3.  Use your standard company credentials.

## Features

### 1. Chat with AI

- Type your question in the message box at the bottom.
- Use the **"+"** button to start a new chat.

### 2. Document Analysis (RAG)

- Drag and drop PDF, basic text, or Markdown files into the chat window.
- The AI will analyze the document and answer questions based on its content.
- **Tip**: Type `#` to see a list of uploaded documents to reference specifically.

### 3. Switching Models

- Click the model name at the top of the chat window (e.g., "GPT-4").
- Select a different model if available (e.g., "GPT-3.5-Turbo") depending on your needs (Speed vs. Intelligence).
